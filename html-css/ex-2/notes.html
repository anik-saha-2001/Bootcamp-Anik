Exercise 2: CSS Variables and Basic Styling

Key Learnings

  • CSS Variables:
    • Defined variables for colors, fonts, and spacing in `variables.css`.
    • Used `var()` function to reference variables in CSS rules.
    • Leveraged CSS variables to maintain consistency and easily update styles.
  • Cascading Styles:
    • Understood the concept of specificity in CSS.
    • Applied more specific selectors to override less specific styles.
  • Default Values:
    • Used default values for variables using `var(--primary-color, black)` syntax.
    • Ensured fallback values in case variables are not defined.
  • Styling Table Rows:
    • Utilized `nth-child` pseudo-class to target specific table rows.
    • Applied styles to alternate rows for better readability.

Challenges Faced

  • Initially struggled with understanding variable scoping and how they are inherited.
  • Encountered issues with specificity conflicts, especially when overriding styles.
