Exercise 4: CSS Units and Responsive Design

Level 1: Centered Layout with Percentage, Viewport Height, and Viewport Width Units

  • `height: 100vh;`: Sets the body height to 100% of the viewport height.
  • `display: flex;`: Enables flexbox layout for centering the content.
  • `justify-content: center;`: Centers the content horizontally.
  • `align-items: center;`: Centers the content vertically.
  • Percentage Widths: Using percentages for width allows elements to adapt to different screen sizes.
  • Max-width: Sets a maximum width to prevent the content from becoming too wide on larger screens.

Level 2: Responsive Layout with Min(), Max(), and Clamp()

  • `clamp()` Function: Creates a flexible font size that adapts to the viewport size.
  • Min and Max Values: Sets minimum and maximum font sizes to ensure readability.
  • Viewport Units (vw): Uses viewport width to scale font sizes relative to the screen size.

Level 3: Continuous Responsiveness with Fluid Grid and Typography

  • CSS Grid: Creates flexible grid layouts that adapt to different screen sizes.
  • `grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));`: Creates columns that automatically adjust to the available width.
  • Media Queries: Target specific screen sizes to apply custom styles.
  • Fluid Typography: Uses techniques like `em` units and `clamp()` to scale font sizes relative to the parent element or viewport size.

By understanding and applying these techniques, you can create web pages that look great and function seamlessly across a wide range of devices and screen sizes.
