HTML Basics and Learning

Semantic Elements

Semantic elements provide meaning to the structure of web content:

  • <header>: Represents introductory content
  • <nav>: Contains navigation links
  • <main>: Specifies the main content of a document
  • <article>: Represents a self-contained composition
  • <section>: Defines a section in a document
  • <aside>: Contains content tangentially related to the main content
  • <footer>: Represents a footer for a document or section

Header Elements

HTML provides six levels of headings: <h1> to <h6>, with <h1> being the most important.

Including CSS and JavaScript

CSS can be included via:

  • Inline styles: Using the style attribute
  • Internal stylesheet: Using <style> in the <head>
  • External stylesheet: Using <link> tag

JavaScript can be included via:

  • Inline scripts: Using the onclick attribute
  • Internal script: Using <script> tag in HTML
  • External script: Using <script src="...">

HTML Validation

To validate HTML:

  • Use W3C Markup Validation Service
  • Check for proper nesting of tags
  • Ensure all tags are closed
  • Use valid attributes

HTML5 Template


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>
