Notes on HTML Examples

1. Three Images with CSS

  • Used the <img> element to add images to the web page.
  • Applied CSS styles like width, height, and border-radius to create circular images.
  • Added hover effects using CSS transitions to enhance interactivity.
  • Utilized placeholder image service (Lorem Picsum) for sample images.

2. Full-Bleed Image with Overlay

  • Implemented a full-bleed header image using CSS background-image and background-size: cover;.
  • Added a gradient overlay using the ::before pseudo-element with linear-gradient.
  • Styled header text to appear above the image while maintaining readability.
  • Optimized the layout for a visually appealing hero section.

3. Responsive Images with Lazy Loading

  • Used the <picture> element to provide responsive images for different screen sizes.
  • Included multiple srcset attributes to specify high-resolution and low-resolution images.
  • Implemented lazy loading for better performance and faster page loading using loading="lazy".
  • Ensured the image adapts to the container using width: 100%; height: auto; in CSS.
