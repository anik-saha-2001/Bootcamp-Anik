Level 2: Notes on Visual Hierarchy

  • Reading Patterns: Utilize familiar reading patterns (like "F" and "Z") to guide the user's eyes naturally through the content.
  • Rule of Thirds: Divide the layout into thirds, placing key elements in these areas for balanced composition.
  • Size and Scale: Larger elements draw attention first. Used large headlines and prominent hero sections to prioritize key messages.
  • Typography: Varied font sizes and weights emphasize hierarchy, making important information stand out.
  • Color and Contrast: High contrast between text and background improves readability and visual impact.
  • White Space: Generous spacing between sections and elements creates a clean and organized layout.
  • Proximity and Repetition: Group related items together and use consistent styling for repetitive elements (e.g., article cards).
  • Grid Layout: Used a CSS grid system for structured and responsive design, enhancing alignment and organization.
  • Responsive Design: Adjusted layout for smaller screens to maintain usability and clarity on all devices.
